# Commands Used (Assignment 2)

## Basic workflow
- git status
- git add .
- git commit -m "message"
- git push -u origin <branch>

## Branching
- git branch
- git checkout -b <new-branch>
- git checkout <branch>
- git fetch origin
- git pull

## Pull Requests
- Open PR on GitHub
- Merge PR (do not delete branch)

## Useful
- git log --oneline
- git remote -v
